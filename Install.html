🤖 안드로이드 설치 방법

1

현재 화면 오른쪽 상단의 점 3개 (⋮) 메뉴 버튼을 누릅니다.

2

[현재 화면을 홈 화면에 추가] 또는 [앱 설치]를 선택합니다.

3

설치 팝업이 뜨면 [추가] 또는 [설치]를 누릅니다.

🍎 아이폰 (Safari) 설치 방법

※ 아이폰은 반드시 Safari 브라우저로 접속해야 합니다. (카카오톡, 네이버 인앱 브라우저 불가)

1

화면 맨 아래 가운데에 있는 공유 아이콘 (네모 안에서 화살표가 위로 나가는 모양)을 누릅니다.

2

메뉴를 위로 올려서 [홈 화면에 추가] (+) 항목을 선택합니다.

3

우측 상단의 [추가] 버튼을 누릅니다.

설치를 마쳤습니다. (로그인 화면으로 이동)
